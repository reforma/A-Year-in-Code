Git Commands

I love the idea of Git Hub its such a cleaver system. I have been using the website version for bits of code from time to time. But i havent really been using it properly. so with the start of this project i decided to do things properly.

Key Commands
  • Git Init

    • Navigate to your Directory with cd path, then use Git Init to start the repro in the directory

  • Git Fetch repro https://

    • This Fetches the file from the git hub server

  • Git Commit

    • This shows the files that have changed since last checked in or commits the changes you have added to the commit with Git Add

  • Git Add

    • This adds files to the commit

  • Git Push

    • This pushes the code to the server

    In some cases I have needed to use Git -set --upstream then the Https:/

  • Git Branch

    • If you just use this it lists the Branches, if you add a new branch name after it will create a new branch

    To delete a branch use git branch -d "Branch Name"

  • Git Checkout

    • This selects whichever branch that you want to modify

  • git ls-tree -r "Branch Name"

    • this will list the files in the branch
